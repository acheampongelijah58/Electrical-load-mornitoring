# Electrical Load Monitoring and Billing Simulator

## Programme
HND Electrical Engineering – Level 200

## Course
EEE 227 (PT)

## Description
This project is a C++ console-based application that simulates electrical load monitoring and electricity billing for a household or small facility.

## Features
- Register electrical appliances
- View all appliances
- Search appliances by name
- Calculate daily energy consumption (kWh)
- Calculate electricity bill based on tariff
- Save and load appliance data from file

## Formula Used
Energy (kWh) = (Power × Hours) / 1000

## Technologies Used
- C++
- File handling (fstream)
- Arrays
- Object-Oriented Programming
- GitHub Version Control

## How to Compile
g++ main.cpp -o electrical

## How to Run
./electrical
